# IN PROGRESS — Website

Agency website for In Progress (IAMSOUND Inc).

## Features
- Dark/light mode toggle (defaults to dark)
- Proximity glow border effect
- Fully responsive/fluid layout
- iOS safe area support
- Anti-scrape email obfuscation
- Contact banner slide-in
